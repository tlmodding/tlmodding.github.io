The Save Editor is a tool that allows you to edit your save file for **Tomodachi Life: Living the Dream**, giving you the ability to change various aspects of your game, such as give you items, money or even change some more advanced data.


### Download

The Save Editor can be found on [GitHub](https://github.com/tlmodding/living-the-dream-save-editor), and is only available for Windows, for now. But can be used on Linux and MacOS with **Wine**.

You have two options for downloading the Save Editor, a self-contained one which doesn't require .NET to be installed, but is a bit larger in size, and a regular one which requires .NET 10 to be installed, but is smaller in size.
