To reverse engineer a Nintendo Switch game, you first need to dump its ExeFS (Executable File System). The ExeFS contains the game's executable code and other essential files.

### Using an Emulator
The easiest way to dump the ExeFS is by using a Nintendo Switch emulator, such as Yuzu or Ryujinx. Both emulators have built-in tools to dump the RomFS and ExeFS from your game.
